1、部署
將 *.yaml 拷貝到 程序的 個人文件夾。 如下：

# 部署位置：
# ~/.config/ibus/rime  (Linux)
# ~/Library/Rime  (Mac OS)
# %APPDATA%\Rime  (Windows)

然後， 進行 重新部署。


詞庫 包括 漢語大詞典、搜狗常用詞庫、中文維基百科、EMOJI（表情）

2、 支持 某些 專有名詞 的 拼音 大寫
  1） 修改 terra_pinyin.custom.yaml
  
3、 支持 emoji 輸入 和 輸出
  1） 按照 emoji.yaml 的方案 ，進行 正常 的 輸入 就 可以 了
  
4、 按照 漢字 的 筆畫 進行 輸入 生僻 漢字
  1） 先 按下 按鍵 “1” 左側 的 “`”或者说 “~” 鍵， 然後 輸入 漢字 的 筆畫 （hspnz）
  2） 利用 的 是， stroke 詞典
